# Context Layer

This directory is the repo-local memory layer for brownfield agent work.

Use it to keep selected, reusable context small and structured:

- `repo-profile.md` for the repo purpose and major areas.
- `commands.md` for the commands that matter.
- `boundaries.md` for forbidden paths and operational limits.
- `conventions.md` for coding and doc conventions.
- `runbooks.md` for recurring workflows.
- `known-errors.md` for common failure modes and fixes.
- `discussions.md` for reusable discussion summaries.
- `repository-intelligence/` for repo-mode-aware classification and knowledge.
- `memory-index.json` for machine-readable selection.

Task-local evidence should stay under `docs/evidence/<task_id>/`.
