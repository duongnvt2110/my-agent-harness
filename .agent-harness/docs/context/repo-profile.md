# Repo Profile

This repository is a scratch harness for agent-assisted development.

The harness scripts control active plans, verification, remediation, review,
and finalization. The docs under `docs/` describe the contract. The live plan
in `docs/exec-plans/active/current.md` is the only active execution source of
truth.

Repository-intelligence artifacts live under `docs/context/repository-intelligence/`
and are selected into task context based on the active repo mode and task mode.
