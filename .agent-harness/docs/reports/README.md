# Reports

Updated: 2026-06-25 16:20

This directory stores reviewable harness quality artifacts generated by the
release gate and benchmark workflow.

## Expected Artifacts

- `release-check.md`
- `benchmark/latest.md`
- `benchmark/latest.json`
- `harness-quality-score.md` when `score-harness.sh` is run manually

## Usage

- `./scripts/benchmark.sh`
- `./scripts/score-harness.sh`
- `./scripts/release-check.sh`
