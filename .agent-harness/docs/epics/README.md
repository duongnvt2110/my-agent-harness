# Epics

Epics are feature and spec intake records for larger harness work.

Each epic is created at runtime under:

```text
docs/epics/<epic_id>/
```

Runtime epic files:

- `epic.md`
- `stories.jsonl`
- `epic-memory.md`
- `integration-contract.md`
- `clarifications.md`
- `progress.md`

The reusable template should not include concrete epic folders.
